97 /*
98  * The TEX, Shareable (S), Cacheable (C) and Bufferable (B) bits define the
99  * memory type, and where necessary the cacheable and shareable properties
100  * of the memory region.
101  *
102  * The TEX, C, and B bits together indicate the memory type of the region,
103  * and:
104  * - For Normal memory, the cacheable properties of the region.
105  * - For Device memory, whether the region is shareable.
106  *
107  * For Normal memory regions, the S bit indicates whether the region is
108  * shareable. For Strongly-ordered and Device memory, the S bit is ignored.
109  *
110  * See the following two tables for setting TEX, S, C and B bits for
111  * unprivileged flash, privileged flash and privileged RAM regions.
112  *
113  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
114  | TEX | C | B | Memory type | Description or Normal region cacheability | Shareable? |
115  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
116  | 000 | 0 | 0 | Strongly-ordered | Strongly ordered | Shareable |
117  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
118  | 000 | 0 | 1 | Device | Shared device | Shareable |
119  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
120  | 000 | 1 | 0 | Normal | Outer and inner write-through; no write allocate | S bit |
121  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
122  | 000 | 1 | 1 | Normal | Outer and inner write-back; no write allocate | S bit |
123  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
124  | 001 | 0 | 0 | Normal | Outer and inner Non-cacheable | S bit |
125  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
126  | 001 | 0 | 1 | Reserved | Reserved | Reserved |
127  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
128  | 001 | 1 | 0 | IMPLEMENTATION DEFINED | IMPLEMENTATION DEFINED | IMPLEMENTATION DEFINED |
129  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
130  | 001 | 1 | 1 | Normal | Outer and inner write-back; write and read allocate | S bit |
131  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
132  | 010 | 0 | 0 | Device | Non-shared device | Not shareable |
133  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
134  | 010 | 0 | 1 | Reserved | Reserved | Reserved |
135  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
136  | 010 | 1 | X | Reserved | Reserved | Reserved |
137  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
138  | 011 | X | X | Reserved | Reserved | Reserved |
139  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
140  | 1BB | A | A | Normal | Cached memory, with AA and BB indicating the inner and | Reserved |
141  | | | | | outer cacheability rules that must be exported on the | |
142  | | | | | bus. See the table below for the cacheability policy | |
143  | | | | | encoding. memory, BB=Outer policy, AA=Inner policy. | |
144  +-----+---+---+------------------------+--------------------------------------------------------+-------------------------+
145  |
146  +-----------------------------------------+----------------------------------------+
147  | AA or BB subfield of {TEX,C,B} encoding | Cacheability policy |
148  +-----------------------------------------+----------------------------------------+
149  | 00 | Non-cacheable |
150  +-----------------------------------------+----------------------------------------+
151  | 01 | Write-back, write and read allocate |
152  +-----------------------------------------+----------------------------------------+
153  | 10 | Write-through, no write allocate |
154  +-----------------------------------------+----------------------------------------+
155  | 11 | Write-back, no write allocate |
156  +-----------------------------------------+----------------------------------------+
157  */
