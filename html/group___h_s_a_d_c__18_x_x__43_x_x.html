Returns
Current raw value of the indexed last sample register
Note
This function returns the raw value of the indexed last sample register and clears the sample's DONE and OVERRUN statuses if set. You can determine the overrun and datavalid status for the sample by masking the return value with HSADC_LS_DONE or HSADC_LS_OVERRUN. To get the data value for the sample, use the HSADC_LS_DATA(sample) macro. The threshold range and crossing results can be determined by using the HSADC_LS_RANGE(sample) and HSADC_LS_CROSSING(sample) macros and comparing the result against the HSADC_LS_RANGE_* or HSADC_LS_CROSSING_* definitions.
