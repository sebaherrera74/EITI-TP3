348 /*
349  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
350  *
351  * The source code that implements the alternative (Alt) API is much
352  * simpler because it executes everything from within a critical section.
353  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
354  * preferred fully featured API too. The fully featured API has more
355  * complex code that takes longer to execute, but makes much less use of
356  * critical sections. Therefore the alternative API sacrifices interrupt
357  * responsiveness to gain execution speed, whereas the fully featured API
358  * sacrifices execution speed to ensure better interrupt responsiveness.
359  */
